# Frontend Projects 

This repository contains multiple frontend development projects built using fundamental web technologies.
Each project focuses on applying core concepts of HTML, CSS, and JavaScript to create functional and user-friendly interfaces.

---

## Projects Included

### 1. KaarigariKart 

KarigariKart is a multi-page e-commerce style website designed to showcase handcrafted and artisanal products.

#### Features

* Multi-page layout (Home, Gallery, Registration, Contact)
* Product showcase with images and descriptions
* Basic form validation using JavaScript
* Clean and simple UI design

### 2. YouTube Clone 

A simplified front-end clone of YouTube focusing on layout and design replication.

#### Features

* Video grid layout
* Thumbnail-based video display
* Navigation bar similar to YouTube
* Responsive-like structure using basic CSS

---

## Learning Outcomes

* Understanding of multi-page website structure
* Practical use of HTML elements and forms
* Styling using CSS and selectors
* Adding interactivity using basic JavaScript
* Replicating real-world UI layouts
