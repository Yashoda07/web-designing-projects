# KaarigariKart

KaarigariKart is a basic e-commerce style website designed to promote, showcase and purchase handcrafted and artisanal products.  
The project focuses on applying foundational web development concepts using HTML, CSS and JavaScript.

---

## Features

- Multi-page website structure (Home, Gallery, Contact, etc.)
- Clean and simple user interface
- Product display using images and descriptions
- Basic form handling and validation
- Interactive elements using JavaScript (alerts, events, DOM manipulation)

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript

## Learning Objectives

This project was created to:

- give artisans and real art admirers to connect on a platform
- Understand the structure of a multi-page website  
- Practice HTML elements like forms, lists, and links  
- Apply CSS styling and selectors  
- Implement basic JavaScript for interactivity
