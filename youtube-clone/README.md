# YouTube Clone

## Project Overview
This project is a front-end clone of the YouTube homepage created using HTML and CSS. The goal of this project was to practice webpage layout design, styling, and component structuring similar to a real-world website interface.

The project replicates the visual structure of YouTube including the header, sidebar navigation, and video grid layout.

## Features
- YouTube-like header with search bar and icons
- Sidebar navigation menu
- Responsive video grid layout
- Video thumbnails with channel information
- Tooltip effects on header icons
- External video links that open in a new tab

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Roboto)
